----------------------------------------------------

セキュリティパックプレミアム（L）は、お客さまのスマートフォンをさまざまな危険やトラブルから守るセキュリティ対策サービスです。

対象プラン対象プラン
----------

|     |     |
| --- | --- |
| プラン | LINEMOベストプラン  <br>LINEMOベストプランV  <br>ミニプラン  <br>スマホプラン |

対象機種対象機種
--------

|     |     |
| --- | --- |
| iPhone | iOS 15.0以上 |
| Android | Android OS10以上 |

申し込み・料金申し込み・料金
--------------

|     |     |
| --- | --- |
| 申し込み | 必要  |
| 月額料 | 770円（初月無料） |

*   ※セキュリティパックプレミアムに初めてご加入の場合、月額使用料が1ヵ月無料でご利用いただけます。

--------

*   SIM単体で回線契約されているお客さまもご加入いただけます。
*   「セキュリティパックプレミアム(L)」に含まれる各サービスは、ご利用中の機種やOSのバージョンなどにより一部ご利用いただけない場合があります。詳細は各サービスの対応機種をご確認ください。
